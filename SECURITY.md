# Security

This project is a research and tooling repository (LaTeX, Python, and static datasets). It does not ship a network service.

If you discover a sensitive data handling issue (for example, accidental inclusion of non-anonymized paths or credentials in a fork or PR), please report it privately via [GitHub Security Advisories](https://github.com/interfluve-wav/dj-metadata/security/advisories/new) for the `interfluve-wav/dj-metadata` repository so maintainers can coordinate a fix before public disclosure.
